# Encapsulation and Constructors in C++

This folder contains Object Oriented Programming programs written in C++ that demonstrate **encapsulation** and different types of **constructors**. These programs focus on how objects are initialised and how data is protected within a class.

The examples are designed using simple logic and real-world scenarios to strengthen conceptual understanding.
